# Radio Widget

> A component that allows selection of a value within a range.

<RadioGroup>
  <RadioButton key="1" value="1" label="Option 1"/>
  <RadioButton key="2" value="2" label="Option 2"/>
  <RadioButton key="3" value="3" label="Option 3"/>
</RadioGroup>
