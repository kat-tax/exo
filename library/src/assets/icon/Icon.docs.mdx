## Platform Support

:::info
For native support you must install [React Native SVG](https://github.com/software-mansion/react-native-svg)

`npm i react-native-svg{:bash}`

And add the following to your **babel.config.js** file

```js
{
  plugins: ['react-exo/babel-plugin-iconify']
}
```
:::
