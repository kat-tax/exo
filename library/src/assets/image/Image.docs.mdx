## Platform Support

:::info
For Android & iOS support you must install [Faster Image](https://github.com/candlefinance/faster-image)

`npm i @candlefinance/faster-image{:bash}`
:::

| Platform | Technology | Functional | Notes |
| -------- | ---------- | :--------: | ----- |
| Web | [React Native Web Image](https://necolas.github.io/react-native-web/docs/image) | ✅ | Uses default platform cache. |
| Android | [@candlefinance/faster-image](https://github.com/candlefinance/faster-image) | ✅ | - |
| iOS | [@candlefinance/faster-image](https://github.com/candlefinance/faster-image) | ✅ | - |
| MacOS | [React Native Image](https://reactnative.dev/docs/image) | ✅ | Uses default platform cache. |
| Windows | [React Native Image](https://reactnative.dev/docs/image) | ✅ | Uses default platform cache. |
