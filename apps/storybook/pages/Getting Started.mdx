# Getting Started

1. Run `pnpm i` to install dependencies
2. Run `pnpm dev` to start the app's dev server
3. Run `pnpm build` to build the app for hosting
4. Run `pnpm storybook` to start this storybook dev server
5. Run `pnpm storybook-build` to build this storybook for hosting
