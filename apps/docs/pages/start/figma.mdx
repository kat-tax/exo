# Start with Figma

You can use the Figma plugin to design components, create themes, choose icons, and export the Figma file as an EXO project.

::::steps

### Run the Plugin

- Go to the [Figma -> React Native](https://www.figma.com/community/plugin/821138713091291738) plugin page
- Press `Open inâ€¦` and select `Playground File`

### Export the Project

- Go to the plugin's `Export` tab
- Press `Download Components`
- Extract the downloaded zip file

### Install Dependencies
  
- Open a terminal in the extracted folder
- Install dependencies with your favorite package manager

:::code-group

```bash [npm]
npm install
```

```bash [pnpm]
pnpm install
```

```bash [yarn]
yarn install
```

```bash [bun]
bun install
```

:::

:::warning
For iOS support, you also need to run:

`cd apps/client/ios && pod install{:bash}`
:::


### Run Development Servers
  
- Open a terminal in your project folder
- Run a script with your favorite package manager

:::code-group

```bash [npm]
npm start
```

```bash [pnpm]
pnpm start
```

```bash [yarn]
yarn start
```

```bash [bun]
bun start
```

:::

:::info
You may now visit the following urls to start developing:
- Web: http://localhost:6206
- Docs: http://localhost:6106
- Storybook: http://localhost:6006
:::
