# Start with Figma

You can use the Figma plugin to design components, create themes, choose icons, and export everything as an EXO project.

::::steps

### Run the Figma plugin

Go to the [Figma -> React Native](https://www.figma.com/community/plugin/821138713091291738) plugin page

[![Figma -> React Native](https://placekitten.com/640/480)](https://www.figma.com/community/plugin/821138713091291738)

### Open the EXO Figma template

Press `Open in...` and select `Playground File`

[![Figma -> React Native](https://placekitten.com/640/540)](https://www.figma.com/community/plugin/821138713091291738)

### Run the plugin

Run the plugin and go to the `Export` tab

[![Figma -> React Native](https://placekitten.com/640/440)](https://www.figma.com/community/plugin/821138713091291738)

### Press the `Download Components` button

Run the plugin and go to the `Export` tab

[![Figma -> React Native](https://placekitten.com/640/320)](https://www.figma.com/community/plugin/821138713091291738)
