# Gesture Utility

> The Gesture Handler Root View is a component that wraps your application's root view to enable gesture handling.

```tsx twoslash
// @noErrors
import React from 'react';
// ---cut---
// 1. Import the gesture utility // [!code focus]
import {GestureHandlerRootView} from 'react-exo/gesture'; // [!code focus]

// 2. Wrap your application with the root view // [!code focus]
export default () => (
  <GestureHandlerRootView> // [!code focus]
    <SafeAreaProvider>
      {/* Your application */}
    </SafeAreaProvider>
  </GestureHandlerRootView> // [!code focus]
);
```
