# Storage Service

> A service that persists key/values to disk. Mainly used for Redux state persistence.

```tsx twoslash
// ---cut---
// 1. Import the storage service // [!code focus]
import {Storage} from 'react-exo/storage'; // [!code focus]

// 2. Create a storage instance // [!code focus]
const store = Storage.init('exo', 1); // [!code focus]

// 3. Set, get, remove, or clear items // [!code focus]
const added = await store.setItem('foo', 'bar'); // [!code focus]
const value = await store.getItem('foo'); // [!code focus]
const removed = await store.removeItem('foo'); // [!code focus]
await store.clear(); // [!code focus]
```
