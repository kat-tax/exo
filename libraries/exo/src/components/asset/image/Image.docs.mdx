## Platform Support

:::info
For iOS support you must install [Faster Image](https://github.com/candlefinance/faster-image)

`npm i @candlefinance/faster-image{:bash}`
:::

| Platform | Technology | Functional | Notes |
| -------- | ---------- | :--------: | ----- |
| Web | [React Native Web Image](https://necolas.github.io/react-native-web/docs/image) | ✅ | Default browser caching |
| iOS | [@candlefinance/faster-image](https://github.com/candlefinance/faster-image) | ✅ | - |
| MacOS | [React Native Image](https://reactnative.dev/docs/image) | ✅ | No cache support |
| Android | [React Native Image](https://reactnative.dev/docs/image) | ✅ | No cache support |
| Windows | [React Native Image](https://reactnative.dev/docs/image) | ✅ | No cache support |
