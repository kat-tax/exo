# Start with Figma

You can use the Figma plugin to design components, create themes, choose icons, and export the Figma file as an EXO project.

::::steps

### Run the Plugin

- Go to the [Figma -> React Native](https://www.figma.com/community/plugin/821138713091291738) plugin page
- Press `Open inâ€¦` and select `Playground File`

<video class="show-dark" src="/videos/start-figma-dark-1.mp4" style={{width: 685}} autoPlay muted loop/>

### Export the Project

- Go to the plugin's `Export` tab
- Press `Download Components`
- Extract the downloaded zip

<video class="show-light" src="/videos/start-figma-light-2.mp4" style={{width: 685}} autoPlay muted loop/>
<video class="show-dark" src="/videos/start-figma-dark-2.mp4" style={{width: 685}} autoPlay muted loop/>

### Install Dependencies

:::code-group

```bash [npm]
npm install
```

```bash [pnpm]
pnpm install
```

```bash [yarn]
yarn install
```

```bash [bun]
bun install
```

:::

### Run Development Servers

:::code-group

```bash [npm]
npm start
```

```bash [pnpm]
pnpm start
```

```bash [yarn]
yarn start
```

```bash [bun]
bun start
```

:::

:::info
You may now access the following dev servers:
- **Native**: *connect a device or simulator to Metro*
- **Web**: http://localhost:6206
- **Docs**: http://localhost:6106
- **Storybook**: http://localhost:6006
:::
