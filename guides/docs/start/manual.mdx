# Manual Installation

You can integrate any EXO primitive, component, hook, or utility in any existing React Native project. Tree shaking ensures that only what you use is included in your final bundle.

::::steps

### Install EXO

First, install `react-exo` as a dependency in your project. Note that any unused export will not be included in your final bundle.

:::code-group

```bash [npm]
npm i react-exo
```

```bash [pnpm]
pnpm i react-exo
```

```bash [yarn]
yarn add react-exo
```

```bash [bun]
bun i react-exo
```

:::

### Use the library

Once installed, you can now import and use any of the available exports as instructed throughout the documentation.

:::warning

Make sure to install peer dependencies as noted throughout each documentation page.

:::

### Install Unistyles

Follow the instructions as documented over at [Unistyles](https://www.unistyl.es/start/setup)

:::info

This step is optional, but offers additional functionality over RN StyleSheets.

:::
