# Start with the CLI

You can scaffold a new React Native EXO project using the command line.

::::steps

### Create the project

:::code-group

```bash [npm]
npm init exo
```

```bash [pnpm]
pnpm create exo
```

```bash [yarn]
yarn create exo
```

```bash [bun]
bun create exo
```

:::

### Run Development Servers

:::code-group

```bash [npm]
npm start
```

```bash [pnpm]
pnpm start
```

```bash [yarn]
yarn start
```

```bash [bun]
bun start
```

:::

:::info

You may now access the following dev servers:
- **Native**: *connect a device or simulator to Metro*
- **Web**: http://localhost:6206
- **Docs**: http://localhost:6106
- **Storybook**: http://localhost:6006

:::
